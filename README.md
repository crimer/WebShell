# Web-Shell
Простое приложение Web-Shell (частичная эмуляция терминала)

Frontend:
- ReactJs

Backend:
- ASP.NET Core Web API
- MSSQL Server
- EntityFrameworkCore
- Serilog

## Запуск
Frontend:
1. Установка необходимых зависимотей - `yarn`
2. Запуск в режиме development - `yarn start`
2. Запуск в режиме production - `yarn build`
